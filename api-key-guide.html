🔑 获取大模型 API Key 傻瓜教程

本教程一步一步带你获取阿里百炼和 MiniMax 的 API Key。不需要编程经验，跟着截图和说明操作即可。

💡 开始前准备：
  • 一个手机号（用于接收验证码）
  • 支付宝或微信（用于实名认证和充值）
  • 大约 10-20 分钟时间

📌 先选哪个平台？

两个平台都可以用，主要区别：

🥇 推荐新手：阿里百炼（通义千问）

  • 优点：国内访问快、中文效果好、有免费额度、文档完善
  • 推荐模型：Qwen-Max 或 Qwen-Plus（性价比高）
  • 适合：第一次尝试、日常使用、中文任务

🥈 备选：MiniMax（海螺 AI）

  • 优点：模型能力强、适合创意写作、角色扮演
  • 推荐模型：abab6.5 或 abab6.5s
  • 适合：需要更强创意能力、已经用过百炼想对比
💰 预算建议：
  • 只想试试：先不充值，用免费额度跑通流程
  • 自己日常用：充值 10-50 元够用很久（看使用频率）
  • 多人测试：充值 100 元左右，注意设置额度提醒

🅰️ 阿里百炼 API Key 获取教程

第 1 步：访问阿里百炼官网

1 打开浏览器，访问 阿里百炼控制台

第 2 步：登录/注册阿里云账号

2 使用手机号或支付宝登录。如果没有账号，按提示完成注册。
📝 提示：需要完成实名认证才能使用 API。按页面提示上传身份证或支付宝认证即可。

第 3 步：进入 API Key 管理页面

3 登录后，在左侧菜单找到 "API-KEY 管理" 或 "密钥管理"，点击进入。

第 4 步：创建新的 API Key

4 点击 "创建新的 API-Key" 按钮。
5 给 Key 起个名字（比如 lobster-ai），然后点击确认。
⚠️ 重要：API Key 只会显示一次！创建后立即复制保存，关闭后就看不到了。如果丢失需要重新创建。

第 5 步：复制并保存 API Key

6 复制显示的 API Key（通常以 sk- 开头），粘贴到安全的地方（比如密码管理器或本地文本文件）。
✅ 完成！你已经成功获取阿里百炼的 API Key。

第 6 步：开通模型服务（如需要）

7 在控制台找到 "模型广场" 或 "模型管理"。
8 找到 Qwen-Max 或 Qwen-Plus，点击开通。
💰 关于充值：
  • 新用户通常有免费额度（约 10-50 元等值 token）
  • 免费额度用完后需要充值才能继续使用
  • 在"费用中心"可以充值，最低一般 10 元起

🅱️ MiniMax API Key 获取教程

第 1 步：访问 MiniMax 开放平台

1 打开浏览器，访问 MiniMax 开放平台

第 2 步：登录/注册账号

2 使用手机号登录。如果没有账号，按提示完成注册。

第 3 步：进入 API Key 管理

3 登录后，点击右上角头像，选择 "API 密钥管理" 或在控制台找到相关入口。

第 4 步：创建 API Key

4 点击 "创建密钥" 按钮。
5 给密钥起个名字（比如 lobster-ai），设置权限（一般选默认即可），然后确认创建。

第 5 步：复制并保存 API Key

6 复制显示的 API Key（通常以 sk- 开头），立即保存到安全位置。
⚠️ 重要：和阿里百炼一样，API Key 只显示一次，关闭后无法再查看，丢失需重新创建。
✅ 完成！你已经成功获取 MiniMax 的 API Key。

第 6 步：充值（如需要）

7 在控制台找到 "充值" 或 "账单" 页面。
8 选择充值金额，完成支付。
💰 MiniMax 价格参考：
  • abab6.5：约 0.1 元/千 tokens（输入）+ 0.3 元/千 tokens（输出）
  • abab6.5s：价格更低，适合日常使用
  • 新用户可能有免费体验额度

📋 获取到 API Key 后做什么？

✓ 回到龙虾 AI 部署页面，在配置文件中填入你的 API Key。
✓ 选择模型：阿里百炼选 qwen-max 或 qwen-plus，MiniMax 选 abab6.5。
✓ 测试连接：运行测试命令，确认 API Key 有效。

❓ 常见问题

Q: API Key 泄露了怎么办？

立即行动：
  1. 马上到对应平台删除/禁用这个 Key
  2. 创建一个新的 API Key
  3. 更新你的配置文件

Q: 免费额度用完了怎么办？

需要充值才能继续使用。建议：

  • 先充值小额（10-50 元）测试
  • 在平台设置"额度提醒"，避免意外超支
  • 如果用量大，可以关注平台的套餐优惠

Q: 两个平台选哪个更好？

建议：

  • 第一次用：选阿里百炼，文档完善、访问稳定
  • 想对比效果：两个都试试，看哪个更适合你的场景
  • 预算有限：对比两个平台的定价，选性价比高的

Q: API Key 填到哪里？

在龙虾 AI 的配置文件中，通常是 .env 文件或部署脚本里的配置项。具体位置参考 部署指南。
