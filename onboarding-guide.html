🗺️ 新手入门路线图

从 0 到 1 使用龙虾 AI 的完整路径。按顺序完成每个阶段，预计总耗时 30-60 分钟。

💡 使用建议：
  • 不要跳步 — 每一步都建立在前一步的基础上
  • 遇到问题先看当前页面的"常见问题"部分
  • 完成每个阶段后休息一下，不要一次性做完

📍 完整路线图

阶段 0：准备 — 确认系统要求、准备账号
阶段 1：获取 API Key — 选择平台、注册、获取密钥
阶段 2：安装环境 — 根据你的系统选择安装方式
阶段 3：配置项目 — 填入 API Key、测试连接
阶段 4：启动服务 — 运行项目、验证功能
阶段 5：日常使用 — 配置自动化、监控状态

📦 阶段 0：准备

阶段 0 预计 5 分钟

开始前需要准备的

  • 一台电脑（macOS / Windows / Linux 均可）
  • 稳定的网络连接
  • 一个手机号（用于接收验证码）
  • 支付宝或微信（用于实名认证）
  • 大约 30-60 分钟完整时间
💡 系统支持：
  • Windows：PowerShell 一键部署，自动检测环境
  • macOS：双击运行，支持 Intel 和 Apple Silicon
  • Linux：支持主流发行版（Ubuntu/Debian/Fedora）

🔑 阶段 1：获取 API Key

阶段 1 预计 10-15 分钟

选择一个平台开始

推荐新手：阿里百炼（通义千问）

  • 国内访问快
  • 中文效果好
  • 有免费额度
  • 文档完善

备选：MiniMax（海螺 AI）

  • 模型能力强
  • 适合创意写作
  • 适合角色扮演
✅ 完成标志：你手上有一个以 sk- 开头的 API Key，并且已经安全保存。

👉 查看详细 API Key 获取教程

💻 阶段 2：安装环境

阶段 2 预计 15-20 分钟

根据你的系统选择

🍎 macOS 用户

  • 安装 Homebrew（如果还没有）
  • 安装 Git、Node.js、pnpm
  • 克隆项目代码

🪟 Windows 用户

  • 启用 WSL2 功能
  • 安装 Ubuntu
  • 在 Ubuntu 中安装 Git、Node.js、pnpm
  • 克隆项目代码

🐧 Linux 用户

  • 使用包管理器安装 Git、Node.js、pnpm
  • 克隆项目代码
✅ 完成标志：在终端中输入 node --version 和 pnpm --version 都能正常显示版本号。

👉 查看详细部署指南 | Windows 用户点这里

⚙️ 阶段 3：配置项目

阶段 3 预计 5 分钟

填入 API Key 并测试

  • 复制环境变量示例文件
  • 编辑配置文件，填入 API Key
  • 选择模型（推荐 qwen-max 或 qwen-plus）
  • 保存配置
⚠️ 注意：
  • API Key 不要分享到公开场合
  • 配置文件不要提交到 Git
  • 如果 Key 泄露，立即到平台删除并重新创建
✅ 完成标志：配置文件已保存，API Key 已填入正确位置。

🚀 阶段 4：启动服务

阶段 4 预计 5 分钟

运行并验证

  • 在项目目录运行启动命令
  • 等待服务启动完成
  • 在浏览器中访问本地地址
  • 测试基本功能
✅ 完成标志：在浏览器中看到龙虾 AI 的界面，并能正常使用。

📈 阶段 5：日常使用

阶段 5 持续进行

配置自动化和监控

  • 配置开机自启动（可选）
  • 设置额度提醒（在 API 平台）
  • 定期检查服务状态
  • 根据需求调整模型和配置
