## 截图注意事项

1. **隐私保护**：API Key 显示页面必须打码或模糊处理
2. **清晰度**：确保文字清晰可读，建议 1280x800 或更高分辨率
3. **标注**：用箭头或红框标注关键按钮和入口
4. **格式**：PNG 格式，文件名按顺序编号

## 如何添加截图

1. 手动访问对应页面并登录
2. 使用截图工具捕获（Mac: `Cmd+Shift+4` 或截图 App）
3. 使用图片编辑工具添加标注和打码
4. 保存到 `site/screenshots/` 目录
5. 更新 `api-key-guide.html` 中的对应图片引用
