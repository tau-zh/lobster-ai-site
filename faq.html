常见问题

关于龙虾 AI 一键部署、模型接入与使用。

这是什么？

龙虾 AI 是一个帮你把 OpenClaw（开源 AI 助手框架）部署到本地电脑的服务。部署完成后，你可以通过微信、QQ、飞书等社交软件与 AI 对话，让它帮你处理各种任务。

和教程有什么区别？

教程需要你自己研究每一步，可能会遇到各种环境问题。我们提供一键部署脚本：填写配置 → 完成支付 → 下载脚本 → 双击运行，自动完成所有配置，减少踩坑时间。

需要自己准备服务器吗？

不需要。龙虾 AI 部署在你自己的电脑上（Windows/macOS/Linux），不需要额外购买服务器。电脑开机就能用，关机就停止。

支持哪些大模型？

支持主流国产大模型平台：

  • 阿里云百炼（通义千问）- 推荐
  • MiniMax（海螺 AI）
  • DeepSeek（深度求索）
  • 智谱 AI
  • 火山方舟（字节）

社交软件支持哪些？

目前支持：

  • 飞书（推荐，接入最简单）
  • QQ
  • 微信（个人号）

部署时可以选一个或多个，后续也可以随时添加。

99 元包含什么？

¥99 是一次性部署工具费用，包含：

  • 专属一键部署脚本（根据你的配置生成）
  • 自动安装所有依赖
  • 配置大模型 API
  • 配置社交软件机器人
  • 部署后 3 天技术支持

不包含大模型 API 使用费（直接交给阿里云等平台，约 ¥10-50/月）。

API 费用贵吗？

不贵。国产大模型价格已经很亲民：

  • 阿里云百炼 qwen-plus：约 ¥0.01-0.02 / 千 tokens
  • MiniMax abab6.5s：约 ¥0.01-0.03 / 千 tokens
  • DeepSeek：约 ¥0.01-0.02 / 千 tokens

日常使用（聊天、写东西、查资料）每月约 ¥10-50。

部署后怎么用？

部署完成后有两种使用方式：

  • Web 界面：浏览器访问 http://localhost:18789
  • 社交软件：如果配置了微信/QQ/飞书，直接在对应的社交软件中与 AI 对话

部署失败怎么办？

部署脚本会自动检测环境问题并给出提示。如遇问题：

  • 检查网络连接（需要访问 GitHub 和模型平台）
  • 查看脚本输出的错误信息
  • 联系智能咨询助手获取帮助

为什么叫"龙虾"和"OpenClaw"？

OpenClaw 是开源项目的名字（Claw = 爪子）。"龙虾"是中文昵称，更容易记住和传播。对外讲"AI 打工人"更容易理解，对内用 OpenClaw/龙虾更方便。
